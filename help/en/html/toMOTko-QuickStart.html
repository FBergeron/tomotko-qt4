Getting Started Quickly

Here are an short and straight-forward guide to get started quickly with toMOTko. You can leave this window open. Just move it on the side to keep it in clear view to follow the instructions.

  • Set the study languages
  • Import existing glossaries
  • Create your own glossaries
  • Set up and start a quiz

Set the study languages

By default, toMOTko's study language are English and Japanese. If you want to learn different or additional languages, you must specify them in the Preferences like this:

  1. Select the Actions menu and click Preferences item.
  2. In the Preferences dialog, in the Languages tab, check (and uncheck) the appropriated languages.
  3. Click Ok.
  4. Set the languages selectors properly (the 2 comboboxes at the top of the screen). The first one must contain your mother language (or source language) and the second one must contain the language you want to learn (or target language).

Import existing glossaries

There are some glossaries already available on toMOTko's website in the Glossaries section. If you see something that fits your needs there, download the glossary files into your computer. To import a glossary into toMOTko, do this:

  1. Select the Actions menu and click Import... item.
  2. Locate a downloaded glossary file and select it.

The content of the glossary file should appear inside your root folder. Repeat the procedure for all the downloaded glossaries.

Create your own glossaries

If none of the available glossaries interest you. You can create your own glossaries like this:

  1. Click the Red Book button (bottom left) to add a new and empty glossary.
  2. To add new words in the glossary, click on it and select the Words tab.
  3. Click the White Sheet button (bottom right) to insert a new word.
  4. Enter the word's data and click Ok.
  5. Repeat as needed.

Set up and start a quiz

To set up a quiz, you must specify which word the quiz will ask. To do that:

  1. Check all the words you want
  2. Check also the parent glossaries
  3. Check also the parent folders, if any.
  4. To start the quiz, select the Actions menu and click Start Quiz.
